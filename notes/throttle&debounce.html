    vuePress-theme-reco zain    2021

    函数节流和防抖

    zain

    # 节流

    如果你室友让你帮忙取快递,你收到请求之后马上去取快递,你室友又有新快递要取,只有等你取完快递回来了,再让你去取下一个快递.这个例子就是节流的思想.

    var flag = true
function throttle(){
  if(!flag){
    return
  }
  flag = false //进入执行函数阶段,设置flag为false,再次执行该函数直接return
  setTimeout(()=>{ //模拟2秒后执行完毕
    console.log("节流")
    flag = true //函数功能执行完毕,设置flag为true,为下次执行该函数做准备
  },2000)
}

    # 防抖

    如果你室友让你帮忙取快递,你收到请求之后,等2分钟,如果2分钟之内没新快递要取,你才出发,如果室友告诉你有新快递要取,你又会等2分钟.这个例子就是防抖的思想.

    var flag = null
function debounce(){
  clearTimeout(flag) //清除上次等待
  flag = setTimeout(()=>{ //开始等待
    console.log("防抖")
  },2000)
}
