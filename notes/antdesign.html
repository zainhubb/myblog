    ant-design-vue 踩坑

    zain

    安装[babel-plugin-import]后按需引入,npm run server提示缺少less-loader

    解决办法:

    1. 修改vue.config.js,添加以下代码

          css: {
        loaderOptions: {
            less: {
                javascriptEnabled: true
            }
        }
    }

    2. 在babel.config.js中将style:true改为style:'css'

    plugins: [
    [
      "import",
      { libraryName: "ant-design-vue", libraryDirectory: "es", style: 'css' }
    ]
  ]
